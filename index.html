安装

需要安装git和Nodejs

Nodejs会自带安装npm，并默认将其加入环境变量

使用命令npm install -g hexo-cli安装hexo，可以加--prefix 路径设置安装路径，否则是默认路径

遇到warn警告可以不管

之后把hexo.cmd所在的路径加入环境变量，应该是直接在安装路径中而不是在bin中

创建仓库

命令为hexo init 本地博客目录

进入本地博客目录，之后两行命令

运行hexo本地服务器

命令为hexo server

之后可以在localhost:4000访问

新建md文件

在本地博客目录使用命令hexo new “test”

会在sources/_post/下生成test.md

托管到github

修改仓库里的_config.yml

安装部署工具hexo-deployer-git --save

之后==每次修改==，都需要在本地博客目录下使用如下命令重新提交

generate是生成静态文件，将md解析成html，生成的html存储在public下

deploy是将文件部署到github上

设置主题

将模板主题clone到themes文件夹里

git clone 模板主题url 目标路径

并且修改_config.yml的theme为主题名称
