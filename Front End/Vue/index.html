Vue

前端框架，擅长于数据操作、表单填写、页面内容根据数据进行渲染、页面元素部分刷新

页面由数据生成，只要数据就可以进行页面视图的渲染

前端发展过程：

  1. 原生JS
  2. jQuery等类库。解决了浏览器大部分兼容性问题
  3. 前端模板引擎。调用模板引擎的方法生成DOM元素渲染页面，而不是直接操作DOM，解决了频繁操作DOM的问题
  4. Vue等前端框架。只渲染变更内容，减少不必要的操作，提高渲染效率，解决渲染整个页面的问题

框架是一套完整的技术解决方案，项目选择框架后难以更改；类库提供的是单一的小功能，对项目的侵入性较小

Vue人气比较高，React使用的人比较多

React除了开发网站还可以开发手机APP，Vue在Weex的帮助下也可以

开发环境版本<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

生产环境mini版本<script src="https://cdn.jsdelivr.net/npm/vue"></script>

Vue实例

在Vue实例中有多个私有属性，分别为

el 挂载点

设置Vue实例挂载点，使实例的作用域为挂载的元素内

可以根据id、class、标签选择器进行挂载，但不可以挂载到和上

data 数据对象

Vue实例的数据定义在data中

当引用这些数据时，普通数据用，数组内数据用，集合内数据用

methods 方法

方法中通过this代表Vue实例，可以获取data中的数据

components 组件

见文章后面创建Vue实例的私有组件

  • Vue实例的其他私有属性还有过滤器filters，指令directives
  • 还有生命周期钩子函数beforeCreate(){}、created(){}、beforeMount(){}、mounted(){}、beforeUpdate(){}、updated(){}、beforeDestroy(){}、destroyed(){}

Vue指令

Vue指令，作为标签的属性，属性值写在Vue实例中

v-text与v-html

v-text，设置标签的文本。属性值写在Vue实例的data中。类似直接把{ { } }写在标签的文本中，但v-text会覆盖标签文本的内容

v-html，设置标签的innerHTML，内容中有HTML结构会被解析为标签，其他与v-text一样

v-on

v-on:事件名="Func()"，v-on:可简写为@，为元素绑定事件，函数Func()写在Vue的methods里，写法Func:function(){ }

v-show

v-show="条件表达式"

满足条件时显示元素，不满足时隐藏元素

无论显示还是隐藏，元素都存在，修改的只是css的display属性

v-if与v-else-if与v-else

用来实现条件渲染，表达式为值为真则渲染元素，值为假则销毁这个元素

v-for

遍历数组进行渲染

渲染结果为

注意，在v-for中，拥有对父作用域属性的完全访问权限。（虽然item是items中的，但是还可以使用items外的parent）

v-model

只用于表单，创建双向数据绑定

<input v-model="msg">，输入文本框的值会改变Vue实例data中msg的值

v-bind

用于绑定属性，可以是src、style、herf、class等标签属性

如<div v-bind:style="{fontSize: size +'px'}"></div>

其中属性的值可以是一个字符串，（引号内部）也可以是JS表达式

  • 可以是字符串加法，可以是函数

  • 可以是对象，渲染结果为该对象的toString（对象属性值改变）

  • 可以是数组，渲染结果是”[a,b,c]”

对布尔值的属性也有效，若值为false的话该属性会被移除

axios

网络请求库，类似$.ajax一样都是Ajax技术的具体实现

axios写在js中，结合Vue一起使用

CDN导包：<script src="https://unpkg.com/axios/dist/axios.min.js"></script>

axios.get()和post

then方法内部传递的回调函数会在请求响应完成的时候触发

这两个回调函数的形参可以自动获取信息，response是服务器的响应内容，err是错误信息

其实是还有一种写法是axios.get().then(function(response){}, function(err){})，即then方法的第二个参数也是一个回调函数，它在响应失败时触发。这种写法和then-catch的区别是，catch不仅可以捕捉响应失败、也可以捕捉响应成功的操作中的逻辑错误，所以建议使用then-catch。

  • then(response=>{})

其中查询参数对象可写作{k1:v1, k2:v2}

其他还有axios.request，head，delete，put等

与Vue配合

axios写在Vue实例的methods中，通过调用方法进行对data的改变

注意，若在then方法参数的函数内使用this，它指代的不是外面的Vue实例

Vue组件

如果将所有内容都写在Vue的实例中，则它会很大

所以出现了组件，用来将Vue实例划分为不同的模块

以后需要什么功能只要调用对应的组件

组件化和模块化的区别：模块化是从代码逻辑的角度划分（如Node），组件化是从UI界面的角度划分
