Vue

JavaScript框架，适用于数据操作、表单填写

页面由数据生成，通过对数据的操作就完成对页面视图的渲染

通过Vue对象将数据和Dom对象绑定，而不是直接操作Dom对象

和类库jQuery对比，框架功能更强大，且需要遵守一套规则

响应式数据驱动

开发环境版本<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

生产环境mini版本<script src="https://cdn.jsdelivr.net/npm/vue"></script>

基础

el挂载点

el的作用是设置Vue实例挂载

Vue实例的作用域为el命中的元素内部

可以根据id选择器、class选择器、标签选择器进行el的挂载

Vue不可以挂载到和上

methods方法

方法中通过this可以获取data中的数据

指令

Vue指令作为标签的属性

内容绑定

v-text，设置标签的文本，作为标签的属性，属性值写在Vue实例的data中。类似直接把{ { } }写在标签的文本中，但v-text不保留标签中文本的内容

v-html，设置标签的innerHTML，内容中有HTML结构会被解析为标签，其他与v-text一样

事件绑定

v-on:事件名=”Func()”或@事件名=”Func()”，为元素绑定事件，函数Func()写在Vue的methods里，写法Func:function(){}

显示隐藏

常用与广告、遮罩层等

v-show=”isShow”且isShow写在data中，或v-show=”age>=18”

Vue单文件组件

参考
基于HTML、且JavaScript只被用来加强特定的视图的Vue项目中，在script标签里使用Vue.component来定义全局组件，紧接着用new Vue({ el: ‘#container ‘})在每个页面内指定一个容器元素

可是，有的项目完全由JavaScript驱动、组件化时，会遇到一些问题，此时选择使用扩展名为.vue的单文件组件
