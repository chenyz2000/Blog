Posted on 2020-07-12 | Post modified: 2020-07-21 | In Front End

JavaScript

是一门编程语言，由浏览器编译并运行

本身和页面动态没有关系，只有和Dom结合，才能使网页动起来

存在形式和放置位置

  • 在script标签里

  • 以文件形式存在，在script里调用

script的type属性默认值是”text/javascript”，可以省略不写

script标签可以写在head里，也可以写在body里

页面会按照从上到下的顺序运行js和显示html

所以script最好写在body里的最下面

语法

每行以分号结尾

使用花括号，不要求缩进

alert()在页面上显示通知

console.log()在控制台打印

变量

用var定义局部变量，无var定义全局变量

数据类型

原始类型：数字，字符串，布尔值

对象类型：数组，字典

原始类型、null、undefined都不可变

null和undefined分别是对象和基本数据类型的空值

数字

parseInt()，parseFloat()将某数字转换成整数、浮点数，不成功则NaN

NaN，表示非数字

Infinity表示无穷大，数字的特殊值

字符串

可以使用单引号，可以使用双引号

s.length

s.trim()，s.charAt(n)，s.substring(from,to)等等，方法类似于python，区间也是左闭右开，也有字符串相加

正则表达式

用于表单验证

布尔值

true false

比较

值相同== !=

类型和值都相同=== !==

或|| 且&&

数组

push(e)尾部添加，pop()尾部删除，unshift(e)头部添加，shift()头部删除

splice(start, deleteCnt, value, …)三个参数分别表示位置、删除数、添加的值

字典

其他

序列化

JSON字符串和对象之间的转化

序列化：JSON.stringify(obj)

反序列化：JSON.parse(str)

转义

url转义

将中文等转为百分号是encode

encodeURI()和decodeURI()

前两个不对字母数字 ~!@#$&*()=:/,;?+’转义，后两个不对字母数字 ~!*()’转义，两组范围不同

字符串转义

eval

eval(“代码”)，相当于Python中eval和exec的合集，既可以编译代码也可以获取返回值

正则表达式

时间处理

全球统一时间UTC，本地时间

语句

语句和Java形式类似

条件语句

循环语句

普通循环不能遍历字典，for each可以遍历字典，遍历的元素e是字典的键

异常

捕获异常

抛出异常

函数

函数类型

  • 普通函数

  • 匿名函数，用来作为其他函数的参数

  • 自执行函数，声明函数并执行，可以用来做封装

作用域

作用域和Python类似

JS中无块级作用域

花括号里的变量可以被外部访问

注：JS6之后有新特性，let关键字可以支持块级作用域

JS采用函数作用域

函数内的变量是内部形参，外部无法访问

JS采用作用域链

函数嵌套时由内向外找变量，内部优先，每个函数有自己的一个作用域链

作用域链在函数执行前已经创建

浏览器解释函数时已经创建了作用域链，此时变量已经声明但未赋值

函数写在哪里作用域就在哪里，跟返回没有关系

闭包

将变量和函数封装到函数里，外部无法获取函数内部的变量

自执行函数就是闭包
