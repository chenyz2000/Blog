JS

Posted on 2020-07-12 | Post modified: 2020-07-12 | In Front End

JavaScript

是一门编程语言，由浏览器编译并运行

本身和页面动态没有关系，只有和Dom结合，才能使网页动起来

存在形式和放置位置

在script标签里

以文件形式存在，在script里调用

script标签可以写在head里，也可以写在body里

页面会按照从上到下的顺序运行js和显示html

所以script最好写在body里的最下面

语法

每行以分号结尾

变量

用var定义局部变量，无var定义全局变量

数据类型

原始类型：数字，字符串，布尔值

对象类型：数组，字典

原始类型、null、undefined都不可变

null和undefined分别是对象和基本数据类型的空值

parseInt()，parseFloat()将某数字转换成整数、浮点数，不成功则NaN，表示非数字

Infinity表示无穷大，数字的特殊值

字符串

可以使用单引号，可以使用双引号

s.length

s.trim()，s.charAt(n)，s.substring(from,to)等等，方法类似于python，区间也是左闭右开，也有字符串相加

正则表达式

用于表单验证

布尔值

true false

比较

值相同== !=

类型和值都相同=== !==

或|| 且&&

数组

push(e)尾部添加，pop()尾部删除，unshift(e)头部添加，shift()头部删除

splice(start, deleteCnt, value, …)三个参数分别表示位置、删除数、添加的值

字典

js根据id定位标签var tag = document.getElementById('id1');

获取标签内容var text = tag.innerText;

重新设置内容tag.innerText="abc";

setInterval("代码语句",1000);
