字符编码

Posted on 2020-05-29 | Post modified: 2020-05-29 | In nec.

编码方案

  • ASCII编码——0~127

  • 扩展编码（加字节，ASCII的超集）

    • 西欧语言：ISO 8859(1-15)——0~255

    • 中文：

      • GB2312——0xA1A1~0xFEFE，双字节表示
      • GBK
      • GB18030

      (GB2312+Big5⊆GBK⊆GB18030)

    • 繁体中文：Big5

    • 日文：Shift_JIS

  • Unicode编码

    • UTF-8:兼容ASCII，1-4个字节变长
    • UTF-16:2或4个字节变长
    • UTF-32:4个字节（32bits）定长

  • ANSI

    windows上非Unicode的默认编码，记事本默认是ANSI存储

    简体中文系统中代表GBK，繁体中文系统中代表Big5，日文系统中代表Shift_JIS

Servlet

Server Applet，是用Java编写的服务器端程序。主要功能在于交互式地浏览和修改数据，生成动态 Web 内容。

狭义的Servlet指Java语言实现的一个接口，广义的Servlet指任何实现了这个 Servlet 接口的类

Java Web开发的基础

与操作系统无关的服务器组件

运行在Servlet容器/应用服务器中

负责与客户端进行通信

即如果用Java开发服务器，客户端发给服务器的数据是通过servlet接受
